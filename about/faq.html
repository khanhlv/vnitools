FAQ

Frequently asked questions about rapidtables.com:

Q: I found a mistake / problem in the website.

A: Please fill in the feedback form below and write down the problem and the page name, so we can fix it.

 

Q: I saw a bad ad. How do I remove it?

A: Ads are set automatically by Google and other ad networks. You should press the little x on the corner of the ad. This will notify the ad network to remove this ad so it will not show again. If this is not a good solution for you, you can install an ad blocking extesion.

 

Q: Chrome vertical scrollbar / scrollbar buttons are missing.

A: Enter URL chrome://flags/ and set "Overlay Scrollbars"="Disabled".

 

Q: I have been blocked when accessing the website.

A: Bad bots protection system was activated. Try to access from other device/browser/IP address. If you use an old browser, upgrade it to a modern browser. If you use a VPN, disable it or use other IP address.

 

Q: I sent an email to rapidtables.com, when will I get an answer?

A: rapidtables.com gets many daily emails and can not answer most of them.

 

Q: Can I link from my website to a page in rapidtables.com?

A: Yes, You may link to any page in rapidtables.com

 

Q: Can I copy content from rapidtables.com?

A: No, the content from rapidtables.com is copyrighted.

 

Q: Do you have a smartphone application of this website?

A: No. most of the pages are adapted for smartphone display.
Select Add to Home Screen to add a shortcut to the web page to your smartphone's home screen.

 

Q: Do you have an application that will enable using the website offline.

A: There is no application of this website. Some pages may be available offline in the future.

 

Q: I want to ask a question about the content.

A: The feedback is only about how to improve the website.

 

Q: How can I publish my ads / buy ad space on rapidtables.com?

A: You can use Google Ads to publish your ads on rapidtables.com.

 

Q: Where are the calculations done?

A: All the calculations are done on the user's local computer.

 

Q: I want to manage rapidtables.com ads.

A: Calculators & Tools gets daily monetization offers. Please avoid sending ad monetization offers.

 

Q: I want to print the page/table.

A: Please avoid printing to save trees.

 

Q: I want to buy rapidtables.com.

A: rapidtables.com is not for sale. If you have to make an offer, please send the amount that you want to offer.
