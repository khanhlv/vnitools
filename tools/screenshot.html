How to take a screenshot

  1. Press the Screenshot button.
  2. Select the window/browser tab that you want to take a screenshot.
  3. Press the Fullscreen button to show the screenshot image with full screen.
  4. Press the Select button to Crop the screenshot image.
  5. Press the Draw button to draw on the sceenshot image.
  6. Press the Text button to write text on the sceenshot image.
  7. Enter sceenshot image file name and type.
  8. Press the Save button to Save the sceenshot image.
  9. Press the Share button to Share the sceenshot image.

Screenshot on Windows PC

Screenshot of screen

  1. Press PrtScn key.
  2. Open Screenshot application and press Ctrl+V keys.

Screenshot of window

  1. Press Alt+PrtScn keys.
  2. Open Screenshot application and press Ctrl+V keys.

Screenshot and save to \Pictures\Screenshots folder

Press ⊞ Win+PrtScn keys.


Screenshot on Mac

Screenshot of screen

Press shift+⌘ command+3 keys.

Screenshot of window

Press shift+⌘ command+4+space keys.

Screenshot with crop

Press shift+⌘ command+4 keys.

Screenshot image file location

The screenshot image file will be saved to the desktop.


Screenshot on Android/Samsung Galaxy

Try one of these options:

  • Press Volume Down and Power buttons.
  • Press Home and Power buttons.
  • Press Back and Power buttons.

Open Gallery app get screenshot file.


Screenshot on iPhone/iPad

Screenshot on iPhone

  1. Try one of these options:

    • Press Volume Up and Side buttons.
    • Press Home and Side buttons.
    • Press Home and Top buttons.
  2. Open the Photos application and select Albums to get screenshot file.

Screenshot on iPad

  1. Try one of these options:

    • Press Volume Up and Top buttons.
    • Press Home and Top buttons.
  2. Open the Photos application and select Albums to get screenshot file.
