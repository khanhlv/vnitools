Phone Call Recorder

Record phone call to audio file (currently with Android only):

Please allow microphone access by clicking the camera icon in the browser's address bar.
Save

Add to Home Screen

To quickly open this page:

  • Windows Chrome: on top right click More > More Tools > Create shortcut
  • iOS/iPhone: on bottom click Share > Add to Home Screen
  • Anroid: on top right click Menu > Add to Home Screen

How to use the voice recorder

  • The microphone access should be enabled by clicking the camera/mic icon in the browser's address bar.
  • Press the Record button to start recording.
  • Press the Pause button to pause recording.
  • Press the Stop button to stop recording.
  • Press audio bar to replay the recording.
  • Press the Save button to save recording to audio file.
  • Press the Share button to share audio file.
