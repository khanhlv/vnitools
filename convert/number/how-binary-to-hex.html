How to convert from hex to binary

Convert each 4 binary digits to hex digit  according to this table:

Binary Hex
0000 0
0001 1
0010 2
0011 3
0100 4
0101 5
0110 6
0111 7
1000 8
1001 9
1010 A
1011 B
1100 C
1101 D
1110 E
1111 F

Example #1

Convert (01001110)2 to hex:

(0100)2 = (4)16

(1110)2 = (E)16

So

(01001110)2 = (4E)16

Example #2

Convert (0100101000000001)2 to hex:

(0100)2 = (4)16

(1010)2 = (A)16

(0000)2 = (0)16

(0001)2 = (1)16

So

(0100101000000001)2 = (4A01)16
